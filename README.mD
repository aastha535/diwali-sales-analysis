# Python_Diwali_Sales_Analysis
## Project overview
this project analyzes customer purchasing behavior during diwali sales using python base exploratary data analysis (EDA)




## objectives
-understand customer dempgraphics and buying patterns
-identify top-performing product categories
-analyze the impact of age, gender,marital status, and occupation on sales
to analyze customer purchasing behaviour during Diwali sales.

##Tools used 
-python
-pandas
-matplotlib
-seaborn
-jupyter notebook

## Key insights
-married women aged 26-35 purchases more products
-IT & healthcare sectors dominate sales
-Top categorioes include  food, clothing, electronics

#source
kaggle Dwali sales Dataset
